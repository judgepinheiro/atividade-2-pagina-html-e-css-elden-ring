# Top 10 Bosses Mais Difíceis de Elden Ring

Projeto desenvolvido para a disciplina de Desenvolvimento Web utilizando HTML e CSS.

A página apresenta um upgrade para a atividade 1 de um ranking com alguns dos chefes mais difíceis de Elden Ring, contendo imagens, descrições, links e estilização com CSS externo.

## o que foi utilizado
    HTML
    CSS

## utilizados no projeto

 Estrutura HTML
 CSS externo
 Seletores de tag, classe e id
 Box Model
 Estilização de listas
 Google Fonts
 Links internos e externos
 Imagens
